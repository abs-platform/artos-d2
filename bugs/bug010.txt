--- Problem description --------------------------------------------------------
Interrupts are enabled inside an interrupt. This is because there are special
types of interrupts that decode into other interrupts. This causes that inside
a IRQ thread (with interrupts enabled), a interrupt handling is requested.

--- Solution -------------------------------------------------------------------
Disable interrupts in msm_summary_irq_handler function in
drivers/gpio/gpio-msm-common.c file.
